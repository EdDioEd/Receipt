# Receipt

Mobile receipt scanning application that digitizes receipts and streamlines expense tracking.

## Overview
Receipt is a mobile-first app that allows users to capture receipts using their phone, store them digitally, and prepare them for expense tracking and reconciliation.

The initial focus is on fast, reliable receipt capture and local management. OCR and backend synchronization will be layered in incrementally.

## Scope (Current)
- Mobile application (iOS & Android)
- Receipt image capture
- Local receipt storage
- Basic receipt metadata (date, merchant, total)

## Out of Scope (for now)
- OCR extraction
- Cloud sync
- Expense approval workflows
- Accounting system integrations
